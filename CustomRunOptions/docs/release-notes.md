# Release notes

## Upcoming release

* The Upcycling Program map event can no longer appear in unmanned runs.

## 1.2.0
Released 13 November 2025.

* Added duo and unmanned runs. Mods can specify the cards and artifacts these runs start with, but all characters and ships have defaults.
* Added an API for mods to register their own custom boot sequence choices.

## 1.1.0
Released 21 July 2025.

* Added options to force or block any boot sequence options.
* Starting a solo run now grants the Solo Run daily run modifier artifact.
* Fixed the mod crashing on controllers.

## 1.0.0
Released 25 June 2025.

* Initial release.
